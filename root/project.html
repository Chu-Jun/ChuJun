Bus Route Tracker

Designed and implemented a bus route tracker utilizing a depth-first search algorithm. The software parses a text file containing bus stops and associated paths. It also offers administrative privileges for path management, including addition and deletion. Additionally, admin can manually add new bus stops while user is able to search for available routes between them.

Language used: Java

Tic-Tac-Toe

Elevated a GitHub repository's tic-tac-toe project. Introduced flexibility to the game by enabling users to customize the dimensions of the tic-tac-toe grid. Enhancing the user experience by innovating and executing a dynamic GUI interface from its original non-GUI state. Users have the capability to engage in multiplayer gameplay, concurrently maintaining a record of their scores, until they choose to end their gaming session.

Language used: Java
